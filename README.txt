Free RR Stalking Bot, Very simple.





Step 1: 

Config.Json webhook_url (discord integration webhook)

Helpful video on grabbing webhook
https://www.youtube.com/watch?v=fKksxz2Gdnc

Step 2:
Config.Json bearer_token (log into a rec room account)
visit https://rec.net/api/auth/session
copy the accessToken Variable
store this inside the bearer_token

Step 3: 
Run it and input a username, and if u want to monitor
room capacity updates along with it!.
